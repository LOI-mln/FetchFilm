# FetchFilm 

A l'aide des cours de 4.10 nous allons developper une application qui categorisent les films et les proposent à l'utilisateur.

Les 3 couches qui seront importantes à réaliser seront : 
 - Le front end qui comprendra l'interactivite la disposition et le rendu que l'on trouvera en arrivant sur l'application (HTML/CSS , JavaScript)

 - Le backend et la base de données qui comprendra la logique métier et applicative (PHP, MongoDB)

 - Integration API : utilisation potentielle de l'API de tmdb pour avoir accès aux ressources des films (nom, affiches, acteurs, notes, dates)


Sur la page d'accueil il y aura : 

    Navbar (Haut) : Logo + Barre de recherche (le plus important).

    Hero Section : Un film "à la une" avec une grande image de fond et un bouton "Détails".

    Le Contenu : Plusieurs rangées de films à scroller (sliders) : les Populaires, les Mieux notés, par Genre 

    Les Tuiles : Chaque film affiche juste l'Image, le Titre et la Note.

    Footer : Crédits de l'API et copyright.
